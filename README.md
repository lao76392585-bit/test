## Getting Started

### Installation

1. Extract the project files to your desired location
2. Ensure all files maintain the folder structure:
   - `index.html` in the root
   - `css/styles.css` for styles
   - `js/script.js` for scripts

### Usage

Simply open `index.html` in your web browser. No build process or dependencies required!

## Customization

### Change Colors

Edit the CSS variables in `css/styles.css`:

```css
:root {
  --primary: #6366f1;
  --primary-dark: #4f46e5;
  --secondary: #ec4899;
  --accent: #14b8a6;
  --dark: #0f172a;
  --light: #f8fafc;
  --gray: #64748b;
  --border: #e2e8f0;
}
```

### Modify Content

Edit sections directly in `index.html`:

- Update service descriptions
- Change portfolio items
- Modify team members
- Update contact information

### Adjust Animations

Fine-tune animation timing in `css/styles.css`:

- Change animation duration
- Modify keyframe percentages
- Adjust delay timing
